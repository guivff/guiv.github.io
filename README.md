# guiv.github.io

Project site for Guiv Farmanfarmaian.

## Local preview

You can open `index.html` directly in the browser, but a local server is better.

If you have Python installed:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages

This repository is configured as a project site and should publish at:

```text
https://guivff.github.io/guiv.github.io/
```
